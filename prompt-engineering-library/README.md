# AI Clipboard Pro (Prompt Engineering Edition)

これは、PCのパワフルな計算能力でAIを動かし、スマホやPC自身からあらゆるテキストを「プロンプトエンジニアリング視点」で変換・整理するためのツールキットです。

## 🚀 主な機能

1. **Analyze (分析)**: 記事から技術コンポーネントを抽出
2. **Enhance (強化)**: 単純な指示をCoT付きの最強プロンプトに変換
3. **Structure (構造化)**: 散乱したメモを綺麗に整形
4. **校正・要約**: 基本機能も完備
